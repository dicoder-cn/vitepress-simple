# v1.0.0 2024-0412

软件的初始版本发布，主要功能如下：

* 支持在线编辑markdown文档
* 支持直接将图片复制粘贴到markdown文档中
* 【多语言】支持多语言配置
* 【导航】支持导航栏可视化配置
* 【侧栏】支持侧栏可视化配置，支持多侧栏配置中，支持自动识别侧栏
* 【搜索】支持搜索配置
